# Phase 1 Checklist

- [ ] App has a stable app id.
- [ ] App can read its installed version.
- [ ] App can detect platform.
- [ ] Manifest URL is defined.
- [ ] Manifest uses HTTPS.
- [ ] Manifest includes latest version.
- [ ] Manifest includes minimum version.
- [ ] Manifest includes download URL.
- [ ] Manifest includes changelog.
- [ ] App compares versions correctly.
- [ ] App handles network errors safely.
- [ ] Optional updates can be postponed.
- [ ] Forced updates are explained clearly.
