## Checkliste Datenschutz (DSGVO)

- [ ] Datenschutzerklärung verlinkt im Feedback-Hub
- [ ] Einwilligung vor Senden von personenbezogenen Daten
- [ ] Opt-out für Screenshot implementiert
- [ ] Opt-out für technische Daten implementiert
- [ ] Kontaktfeld: Hinweis was mit E-Mail passiert
- [ ] Screenshots: sensible Felder maskiert (Passwörter, Kreditkarten)
- [ ] Datenübertragung nur via HTTPS
- [ ] Speicherort der Daten dokumentiert (EU / non-EU)
- [ ] Aufbewahrungsfrist definiert (Empfehlung: max. 2 Jahre)
- [ ] Löschfunktion implementiert (DSGVO Art. 17)
- [ ] iOS Privacy Manifest falls iOS-App
- [ ] Datenpannen-Meldepflicht dokumentiert (DSGVO Art. 33, 72h)
- [ ] Auftragsverarbeitungsvertrag mit Hosting-Anbieter
