# 08. Platforms

## macOS

Common formats are `.dmg`, `.pkg` and signed `.zip`. Important topics are code signing, notarization and Gatekeeper.

## Windows

Common formats are `.exe`, `.msi` and MSIX. Important topics are code signing, SmartScreen and installer permissions.

## Linux

Common formats are `.AppImage`, `.deb`, `.rpm`, Flatpak and Snap. Distribution differences matter.

## Mobile

Binary updates usually go through app stores. Your own backend can still provide minimum version checks, feature flags and content updates.
